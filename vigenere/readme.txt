This is a Vignere cipher encrypter and decrypter

It can be compiled using gcc

To run:

./program_name e/d input_file output_file keyword

program_name is the program name

e is for encrypt
d is for decrypt

input_file is the name of the file you want to encrypt

output_file is the name of the file you want to put the data in

**input_file and the output_file cannot match**
**output_file will delete all the data in the destination**

keyword is the key to encrypt the cipher on

**keyword has to be lowercase with no special characters**
